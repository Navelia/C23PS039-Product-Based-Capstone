}

    • Update Profil

  • url
    • /profil/:id => ":id" diganti dengan uuid user yang ingin di update
  • Method
    • PATCH
  • Request Body 
    • {
  Gunakan form-data pada body
        Key               |  type  |  Value
    >   file                 file    "pilih gambar"
  Gambar yang diunggah harus < 5 mb & memiliki format JPG/JPEG/PNG
}
  • Response 
    • {
  Status 200 >> Berhasil      >> Menampilkan Response
      {
        "msg": "Profile Created Successfully",
        "url": "XXXXXXXX"
      }
  Status 404 >> uuid Salah >> msg: "User not found"
  Status 400 >> Tidak terdapat file >> msg: "No File Uploaded"
  Status 422 >> Tidak menggunakan format JPG/JPEG/PNG >> msg: "Invalid Image"
  Status 422 >> Gambar lebih dari 5 MB >> msg: "Image must be less than 5 MB"
  Status 500 >> Gagal Upload >> msg: "Failed to upload file to Google Cloud Storage"
  Status 500 >> Server eror >> msg: "error.message"
}
